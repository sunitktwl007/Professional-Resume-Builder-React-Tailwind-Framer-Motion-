## Core Resume Building

### 1. Personal Information Section

**What it does:**
- Collect basic personal details
- Upload and display profile photo
- Store complete contact information
- Add professional links

**Fields:**
- Full Name (required)
- Email (required)
- Phone Number
- Website/Portfolio
- Street Address
- City
- State
- Country
- Zip Code
- LinkedIn Profile
- GitHub Profile
- Profile Photo (image upload)

**How to use:**
1. Click the 👤 Personal Information section
2. Upload your professional photo (JPG, PNG)
3. Fill in your contact details
4. Photo appears in all templates
5. Changes update instantly in preview

**Tips:**
- Use a professional headshot (300x300px recommended)
- Keep contact information up to date
- Add social links for easy verification

### 2. Professional Objective/Summary

**What it does:**
- Add brief career objectives
- Write detailed professional summary
- Appears in most template headers

**Fields:**
- Objective (short version, 1-2 lines)
- Summary (detailed version, paragraph)

**How to use:**
1. Click the 🎯 Objective/Summary section
2. Write your professional objective
3. Add a detailed summary
4. Appears prominently in the preview
5. Can vary by template

**Examples:**
```
Objective: "Seeking a challenging role as Software Engineer 
in a dynamic organization to utilize technical skills and 
contribute to company growth."

Summary: "Results-driven Full Stack Developer with 5+ years 
of experience in building scalable web applications. 
Proficient in React, Node.js, and MongoDB with a track 
record of delivering high-quality solutions under tight 
deadlines."
```

### 3. Education Section

**What it does:**
- Add educational background
- List degrees, institutions, and dates
- Include coursework and achievements

**Fields per Entry:**
- Institution Name (e.g., "MIT")
- Degree (e.g., "Bachelor of Science")
- Field of Study (e.g., "Computer Science")
- Start Date (month/year)
- End Date (month/year)
- Description (coursework, GPA, honors, etc.)

**How to use:**
1. Click the 🎓 Education section
2. Click "+ Add Education"
3. Fill in all education details
4. Click "+ Add Education" for multiple degrees
5. Drag to reorder entries (if supported)
6. Delete entries by clicking the trash icon

**Example Entry:**
```
Institution: Massachusetts Institute of Technology
Degree: Bachelor of Science
Field: Computer Science
Start Date: September 2018
End Date: May 2022
Description: "GPA: 3.8/4.0 | Dean's List | Relevant 
coursework: Data Structures, Algorithms, Artificial 
Intelligence, Computer Networks"
```

### 4. Professional Experience Section

**What it does:**
- List work history
- Describe job responsibilities
- Highlight key achievements
- Show progression and growth

**Fields per Entry:**
- Job Position (e.g., "Senior Developer")
- Company Name
- Start Date (month/year)
- End Date (month/year)
- Currently Working (checkbox)
- Description (responsibilities and achievements)

**How to use:**
1. Click the 💼 Professional Experience section
2. Click "+ Add Experience"
3. Enter position and company details
4. Check "Currently working here" if applicable
5. Write bullet points or paragraphs about achievements
6. Add multiple experiences
7. They appear in reverse chronological order

**Example Entry:**
```
Position: Senior Full Stack Developer
Company: Tech Company Inc.
Start Date: January 2021
End Date: Present
Currently Working: Checked
Description: "• Led development of microservices 
architecture reducing load time by 40%
• Mentored 3 junior developers
• Implemented CI/CD pipeline increasing deployment speed
• Improved application performance by 35%"
```

## Export & Storage

### PDF Export

**What it does:**
- Convert resume to professional PDF
- Maintain formatting and styling
- Ready for printing

**How to export as PDF:**
1. Click Download button in header
2. Select "PDF" from dropdown
3. PDF will download automatically
4. Name format: `YourResumeName_[timestamp].pdf`

**Tips:**
- PDF captures current template styling
- Colors and images included
- Optimized for printing (A4 size)
- Compatible with all PDF readers

### JSON Export

**What it does:**
- Save resume as structured data
- Backup for later import
- Portable data format

**How to export as JSON:**
1. Click Download button in header
2. Select "JSON" from dropdown
3. File downloaded to your computer
4. Can be re-imported later

**Use cases:**
- Backup important resume
- Transfer between computers
- Edit raw JSON if needed
- Version control

### Import Resume

**What it does:**
- Load previously saved resume
- Restore from backup
- Import shared resume files

**How to import:**
1. Click Download button in header
2. Select "Import" option
3. Choose `.json` file from computer
4. Imported resume appears in tabs
5. Auto-switches to new resume

**Supported formats:**
- `.json` files from export
- Must be valid resume JSON
- Preserves all data including photos

## Multi-Resume Management

### Create New Resume

**What it does:**
- Create multiple resume versions
- Manage different versions
- Easy switching between resumes

**How to create:**
1. Click "+ New" button in header
2. Enter resume name (optional)
3. New resume appears in tabs
4. Auto-switches to new resume
5. Starts with empty template

**Use cases:**
- Different job applications
- Different industries
- Different experience levels
- Chronological vs. functional versions

### Resume Tabs

**Features:**
- Shows all created resumes
- Click to switch between them
- Current resume highlighted in blue
- Shows resume name
- Tab scrollable on mobile

### Resume Operations

#### Rename Resume
1. Click Download menu in header
2. Select "Rename" option
3. Enter new name in modal
4. Click "Save"
5. Resume tab updates

#### Duplicate Resume
1. Click Download menu
2. Select "Duplicate" option
3. Creates copy with "(Copy)" suffix
4. All data copied over
5. Can modify independently

#### Delete Resume
1. Click Download menu
2. Select "Delete" option
3. Confirm deletion
4. Resume removed permanently
5. Switches to another resume

## Tips & Best Practices

### Resume Writing Tips

1. **Use Action Verbs**
   - Led, Developed, Implemented
   - Managed, Designed, Deployed
   - Increased, Improved, Optimized

2. **Quantify Achievements**
   - "Increased sales by 40%"
   - "Reduced load time from 5s to 2s"
   - "Managed team of 8 engineers"

3. **Keep It Concise**
   - 1-2 pages for entry-level
   - 1-2 pages for experienced
   - Bullet points, not paragraphs
   - Focus on impact

4. **Tailor Content**
   - Match job description keywords
   - Prioritize relevant experience
   - Highlight relevant skills
   - Focus on achievements

### Template Selection Tips

1. **For Tech Roles**
   - Modern, Modern Dark, Geometric
   - Shows technical proficiency
   - Contemporary design

2. **For Creative Roles**
   - Creative, Gradient, Accent
   - Displays design sensibility
   - Eye-catching layouts

3. **For Corporate Roles**
   - Professional, Corporate, Classic
   - Traditional appearance
   - Formal presentation

4. **For Entry-Level**
   - Minimal, Minimalist, Modern
   - Clean presentation
   - Focus on content

### Export Tips

1. **Before PDF Export**
   - Check all content
   - Verify formatting
   - Review spelling
   - Check spacing

2. **PDF Naming**
   - Use professional names
   - Include version/date
   - Example: `FirstName_LastName_Resume_2024.pdf`

3. **Sharing Tips**
   - Always use PDF for sharing
   - Keep JSON backup
   - Version your backups
   - Update regularly
